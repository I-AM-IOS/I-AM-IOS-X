README — Genesis/JSONFlow IDE
◈ Genesis / JSONFlow IDE
Sovereign OS · DDC Infrastructure · v1.0.0

This is a 3-stage deterministic compilation pipeline IDE, styled after Remix IDE. Describe an app in natural language and watch it compile all the way to runnable code in 9 languages.

◈ Stage 1 — Genesis Module
Natural language → deterministic JSON module. LLM enforces 11 canonical invariants. Uses Ollama locally.
◈ Stage 2 — JSONFlow IR
Module → JSONFlow Intermediate Representation. Pure deterministic compiler (no LLM). Produces deriveState, scheduleNextAction, signal functions.
◈ Stage 3 — Code Output
JSONFlow → Python, JS, TS, Rust, Go, Java, C#, Swift, Kotlin.
